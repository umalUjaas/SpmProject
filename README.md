## Setup Instructions

### Local Development

1. **Requirements**:
   - PHP 7.4 or higher
   - Web server (Apache, Nginx, or PHP built-in server)

2. **Using PHP Built-in Server**:
   ```bash
   php -S localhost:8000
Then visit http://localhost:8000 in your browser.


3. Using Apache/XAMPP:

Place all files in your htdocs directory
